# 📊 Interactive Sales Analysis Dashboard (Excel)

## 📌 Overview
This project presents a **Sales Analysis Dashboard** built using Microsoft Excel to analyze business performance across multiple dimensions such as sales, profit, quantity, and customer distribution.  
The dashboard transforms raw data into **interactive, visual insights** that support **data-driven decision making**.

## 🎯 Objectives
- Analyze overall business performance
- Identify top and bottom performing products
- Compare sales and profit across categories and subcategories
- Track customer distribution geographically
- Provide interactive filtering for dynamic analysis

## 📊 Key Metrics (KPIs)
| KPI | Value |
|-----|-------|
| Total Sales | $9,894,184 |
| Total Profit | $1,430,429 |
| Average Discount | 16% |
| Total Quantity Sold | 37,873 |
| Profit Margin | 14% |

## 📈 Dashboard Features
1. **Category Analysis**
   - Total Sales by Category
   - Total Profit by Category
   - Total Quantity by Category
2. **Subcategory Analysis**
   - Profit distribution by subcategory
   - Sales vs Profit comparison
3. **Product Performance**
   - Top 10 Products by Sales
   - Bottom 10 Products by Sales
4. **Geographic Analysis**
   - Total Customers by State (Map Visualization)
5. **Interactive Filters**
   - Year filter (2015–2018)
   - Category filter
   - Quarter filter

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers for interactivity
- Data Cleaning & Preparation
- KPI Cards Design
- Data Visualization Best Practices

## 💡 Key Insights
- Technology category leads in total sales
- Office Supplies has the highest quantity sold
- Certain subcategories generate high sales but low profit (potential margin issue)
- Clear variation in performance across different states
- Significant gap between top and bottom performing products

## 🚀 How to Use
- Open the Excel file
- Use slicers (Year, Category, Quarter) to filter data
- Analyze charts and KPIs dynamically
- Explore product-level insights
